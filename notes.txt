Responsive Web Design
    - Design that works well on any device, any resolution.
    - works well - easy to read and navigate without having to zoom in and out.
    - Ingredients of responsive design
        1) Fluid grid: all layout elements are sized in relative units, such as percentages, instead of absolute units like pixels.
        1) Fluid grid: all layout elements are sized in relative units, such as percentages, instead of absolute units like pixels.
        2) Flexible images: are also sized in relative units.
        3) Media queries: allow us to specity different CSS style rules for different browser widths
    - respond.js - script that enables the browser to understand and execute media queries which we need for responsive webdesign
